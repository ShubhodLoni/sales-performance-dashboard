# Sales Performance Dashboard

## Project Overview

This project analyzes sales, profit, customer segments, and product performance using Power BI.

The dashboard provides insights into:

- Sales trends over time
- Regional performance
- Category-wise profitability
- Top-selling products
- Impact of discounts on profit

## Tools Used

- Power BI
- DAX
- Excel

## Key Metrics

- Total Sales: $2.30M
- Total Orders: 5,009
- Total Profit: $286.40K
- Profit Margin: 12.47%

## Key Insights

- West region generated the highest sales.
- Technology category delivered the highest profit.
- High discount levels negatively affected profitability.
- A small set of products contributed significantly to revenue.
